# p5js-pong-game

Take the html file in the pong folder and run it in any browser other than Chrome. Until/if I upload it to a server, it won't run properly. 

To move the paddles:
Left paddle: UP -- 'w' Down -- 's'
Right paddle: UP -- 'UP_ARROW' DOWN -- 'DOWN_ARROW'
